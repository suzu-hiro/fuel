marginの相殺 TOP

基本設定

.viewer {
  display: block;
  padding: 10px;
}
.item {
  width: 150px;
  background: #999;
}

隣接する要素のマージンの相殺

20px と 30px が設定されているが、マージンは合計で50px ではなく、相殺されて30pxになる

.item1
.item2
.item1 {
  margin: 20px 0;
  height: 30px;
}
.item2 {
  margin: 30px 0;
  height: 30px;
  height: 30px;
}

入れ子の要素のマージンの相殺

子要素に兄弟がない場合は、親子の要素は重なり、マージンは相殺される

.item2
.item1 {
  margin: 30px 0;
  width: 200px;
  background: red;
}
.item2 {
  margin: 20px 0;
  height: 30px;
}

入れ子の要素のマージンが相殺されないケース 1

子要素の兄弟

.item1
.item2
.item1 {
  margin: 30px 0;
  width: 200px;
  background: red;
}
.item2 {
  margin: 20px 0;
  height: 30px;
}

入れ子の要素のマージンが相殺されないケース 2

親要素にpadding

.item2
.item1 {
  margin: 30px 0;
  padding: 10px;
  width: 200px;
  background: red;
}
.item2 {
  margin: 20px 0;
  height: 30px;
}

入れ子の要素のマージンが相殺されないケース 3

親要素にborder
子要素のborderは問題なし

.item2
.item1 {
  margin: 30px 0;
  border: 1px solid green;
  width: 200px;
  background: red;
}
.item2 {
  margin: 20px 0;
  height: 30px;
}

入れ子の要素のマージンが相殺されないケース 4

親要素が flex
子要素が flex は問題なし

.item2
.item1 {
  display: flex;
  margin: 30px 0;
  width: 200px;
  background: red;
}
.item2 {
  margin: 20px 0;
  height: 30px;
}
